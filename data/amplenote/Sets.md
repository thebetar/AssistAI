# Sets

A set of numbers is a list of numbers. When a number X is part of a set Y this can be written out as X ∈ Y

### Subset

A subset of a set is a set of numbers that originate from a set when something is a subset of a set for instance X is a subset of Y this can be written out as $X ⊆ Y$. If multiple subsets come from one set this is called a family

### Element of

Within Mathematics the ∈ sign is used to mark that a is an element of b when used like this $a ∈ b$

### Proper subset

A proper subset of a set is a set containing all entries but which is not the same as the parent set.

### Intersection

When referring to the intersection of two sets the ∩ character is used so if there are two sets and you want to equal the intersection to it you could do $x ∩ y = z$. When doing this the ∅ symbol also becomes very handy which is the symbol to represent an empty set. So if there are no intersecting entries in two sets this would look like $x ∩ y = ∅$

### Empty

Like mentoined above an empty set is annotated with a $∅$

### Cardinality

Cardinality of a set is like the length of an array. It describes the amount of entries in a set.

### Power set

A power set is all possible subsets of a set. This is written like this $P(A)$ where A refers to the set.

### Cross products

A cross product is a set that is created from two sets with every possible combination in the new set annotated with normal braces (). This looks like this $x = {1,2}$, $y = {a,b}$ where\
$x \* y = {(1a), (1b), (2a), (2b)}$.

### Sum

Sets can also be summed. When doing this it needs to be noted that a set contains only unique values so the sum of set {1, 2, 3} and {3, 4, 5} will become {1, 2, 3, 4, 5} this is notated as $x ∪ y = z$ or more specifically ${1, 2, 3} ∪ {3, 4, 5} = {1, 2, 3, 4, 5}$. The symbol used for this is called union

### Difference

In math the difference between two sets can be done to find out what entries are unique to a set and not in the other set this is annotated with a **-** or \*\*\*\*. This would look something like ${1,2,3,4} - {2,3,4,5} = {1}$

### Complement of set

When comparing two sets one set can contain more entries than the other. To annotate this difference a complement of set can be used, for instance if a set $x = {1,2,3}$ and a set\
$y = {1,2,3,4,5}$ the complementary set of $x$ would be $x’ = {4,5}$ as compared to $y$

## Countable

A set is countable if the number of elements is finite and a real number.

## Infimum and suprema

The infimum is the largest number that is smaller or equal to every number in a set so of a list of numbers $\[2,3,4\]$ it would be smaller than 2. The suprema is the other way around this is the smallest number which is equal or larger than every number in the set in this case this would be higher than 4.

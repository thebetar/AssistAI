# JavaScript

JavaScript is with Python the most popular language currently out there. It was designed to be used on the web but since the creation of node.js, which can take javascript code and run it in a non browser environment to create servers, programs or anything else it has been used for all kinds of things. The strength of javascript is that for a lot of different solutions you can stick with just one language. This is very good for beginners since they will be able to make lots of things while only sticking with one language and mastering that one. In the current ecosystem of so many languages this has become the preferred choice for many.

This however is also a big downside of javascript since a lot of developers also stay with javascript since they do not have a reason to switch, since everything can be made with javascript however it is not the optimal solution. This is why a lot of things that should not be built in javascript are built in javascript. For companies however it keeps being interesting to have developers who can work on a whole range of products while not having to switch context too much.

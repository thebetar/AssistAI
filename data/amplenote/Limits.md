# Limits

A limit within math is to show what to max value of a function. This works with functions that contain a variable and the result of the function with the variable is limit that it can reach with a value that i equal to that variable. This looks something like this for a simple example

$lim \\above{0pt} x→1$ $x^2-1 \\above{1pt} x-1$ = 2

This equation states that if x = 1 than the outcome of the function will not be higher than two. In this instance this speaks for itself since the result is actually 2. An even simpler example that is correct looks like this

$lim \\above{0pt} x→20$ $x \\above{1pt} 2$ = 10

Now an important question is why do you not just calculate the result and be done with it. This is because of the concept of **infinity** this way you can have a function that has infinite iterations but indicate that according to your function it will never reach a certain value a simple example of this looks like this

$lim \\above{0pt} x→∞$ $1 \\above{1pt} x$ = 0

This is because infinity starts at 1 and keeps going but no matter how far you go you will never reach 0 even at things like $1 \\above{1pt} 1.000.000.000$ and as wel all know a billion is a pretty large number.

since the value will also not fall below zero

To calculate this a value most likely that when it starts it returns undefined, that is why it is good when encountering math problems like this

$lim \\above{0pt} x → 2$ $x^2-4 \\above{1pt} x - 2$ = ?

to see what result you get with the max value of in this instance `2` and then try a fraction lower something like `1.999` and see what value it is approaching. If this does not provide enough information more digits can be added after the decimal point. It is also a good idea to do this the other way so in this case `2.001` since sometimes a line does not converge on a single point and the limit function does not apply.

A limit is generally declared with a constant value as it’s limit and not with greater than or less than.

Fuzzy finder is one of the most popular libraries used to find files in a system. It uses a very efficient string search in the file system to find files. It can also be chained to send it's input to and editor like `vim` or to only search in a specific folder with `find`

To use fuzzy finder install it via [https://github.com/junegunn/fzf](https://github.com/junegunn/fzf) 

To use it run 

```
fzf 
```

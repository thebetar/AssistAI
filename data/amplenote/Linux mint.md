Linux mint has recently gained a lot of popularity since it does everything well Ubuntu also does well. It has high usability and reliability. Another advantage is that it looks a lot like the windows layout which makes it feel familiar to people migrating from windows systems.

Linux mint is debian based.

What Linux mint has recently done to give it a favourable spot over [Ubuntu](Ubuntu.md) is to stay with the `apt` package manager which currently makes it more performant on slower systems and it offer all the security updates for free. One thing to consider however is that because it is still less popular than Ubuntu that the community and guides are still lower in number. However I would argue that if you know your way around this would be a better distro to try as a beginner.

# Python

Python is nowadays the preferred language thaught in many university and other schooling curriculums. This is because python is very beginner friendly. This is because two main things: it is a high level language which means you do not have to think about memory management which the language does for you, and that it is an indentation based language. This second pro is why it is the preferred language to learn first over javascript since it forces the developers to have correct indentation in their files and not make a mess out of it.

Python is used nowadays mostly for data science purposes this is because there are many libraries that are written in C or C++ that can be called from python code. This combines the easy of use of python with the performance of lower level languages. However Python is also used a lot nowadays to create webservers, computer applications and a whole lot of other things.

# Summation

The summation symbol in math is written like this: ∑. This symbol is used to create a for loop in math. The number on the bottom value is the starting value and the number on the top is the amount of times the summation is run. Finally the function behind it is run everytime an iteration is performed and the results are summed up together, hence why it is called a summation.

When notating a summation sometimes the iteration value is also notated as a lower n for conveniency.

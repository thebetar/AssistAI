My idea is to create a local CLI tool which can use local documents to give answers. This is useful for instance to find files or to recollect some knowledge you used in the past but forgot.

\

Things to use:

- Ollama for open source serving

    - TinyLLama for local testing on non GPU desktop

    - GPT4All ([https://github.com/nomic-ai/gpt4all](https://github.com/nomic-ai/gpt4all))

- Langchain ([https://python.langchain.com/docs/get_started/quickstart](https://python.langchain.com/docs/get_started/quickstart))

- Python 

- Golang ([https://go.dev/](https://go.dev/))\*

- Electron ([https://www.electronjs.org/](https://www.electronjs.org/))

# NeoVim

NeoVim is the evolution of Vim. While Vim introduced a lot of good practises with the vim motions the decision to stick with vim script as an extension language instead of switching to a more mature language has recently made a lot of people switch from vim to neovim. So what is neovim. It is an editor which uses Lua instead of vimscript which is a more mature language and this also makes it easier to write exensions for neovim. Aside from that the community behind NeoVim have also achieved better performance with neovim than with actual vim.

The biggest advantages are this

- The language for extensions is in Lua which is a mature language

- The code is better written so neovim performs better

- Neovim has an active community with basically infinite plugins to make your experience perfect

## Vim motions

If you do not know about the Vim motions yet I would suggest going back to my section on Vim which goes over the vim motions. A lot of the vim motions are also used in neovim while some are also omitted from use.

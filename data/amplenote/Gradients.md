# Gradients

A gradient is also referred to as the slope of an angle. This is when you take two points in a graph and calculate how steep the slope is of the line. Simply said how many points on the Y axis does it go up for every point on the X axis. The is calculated by taking a point where the X and Y intersect nicely with the line and a second point where X and Y intersect and then the distance in X and Y will be divided by each other with $y\\above{1pt}x$.

Aside from calculating the slope between two points there also exists a clever trick to calculate the slope on one single point which is done by taking that point $x, y$ where $f(x) = y$ and then taking an imaginary real number lets say $Δx, Δy$. Now we can start calculating what the slope is between these points by using ${f(x +Δx) - f(x) \\above{1pt} Δx}$ which will result in the slope. Then after creating this we can start reducing the slope to 0 to find out what the slope is at point $x$.

This reducing distance to zero is sometimes also called $d$ which results in the above calculation being shorthanded to $dy\\above{1pt}dx$.

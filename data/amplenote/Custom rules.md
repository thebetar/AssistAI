# Custom rules

An important part of using ESLint is writing custom rules. This is very important when working in teams and you want to enforce specific code standards. This can be done by writing custom rules for eslint. The eslint parser uses AST to select specific code. You can use the following web application to explore AST [https://astexplorer.net/](https://astexplorer.net/)

What is retracing?

An image can be represented with a matrix where each element in the matrix is the color of the pixel in the image it corresponds to

There are multiple possibilities of bits per pixel that can be represented in the matrix for how many colors are supported. If there is only 1 bit it corresponds to black or white with a 0 or a 1 this goes up all the way to 32 bits which supports 16.777.216 colors.

Images mostly work with vectors on a 2D plane which determines where the pixel or shape is drawn.

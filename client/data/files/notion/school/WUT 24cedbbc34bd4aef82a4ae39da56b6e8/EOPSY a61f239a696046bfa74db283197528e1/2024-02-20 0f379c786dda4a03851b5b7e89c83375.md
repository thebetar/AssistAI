### Definitions

- Operating system: distributes resources from the computer and displays a virtual machine that is easier to handle than directly interacting with the hardware
- Resource: all hardware components or software modules that a system has access to
    - hardware components, actual physical resources
        - CPU time
        - working memory
        - external devices
    - Software modules, interpretation of a software resource, like binary on a harddisk
        - files
        - buffers
        - semaphores

Manageing resources means the following:

- Track resources
- Provide metadata of resource
- Alloc resources
- Recover resources

A computer is only a computer if it can read and store it’s instructions in memory and run it. - Von Neumann

A program that can launch other programs is called a software loader. This is also called a loader and is a prototype of an operating system. It is usually used to load the programs needed to start a computer.

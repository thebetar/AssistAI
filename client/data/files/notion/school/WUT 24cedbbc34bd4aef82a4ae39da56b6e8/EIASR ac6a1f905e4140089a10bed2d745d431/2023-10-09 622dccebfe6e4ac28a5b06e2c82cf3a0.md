For pattern recognition multiple steps are done

- The pattern
- Normalisation
- Normalisation pattern
- Template

Within these steps machine learning can be used to see if the result fits with the input if not the system can learn for the next iteration.

Within pattern recognition bitmaps are used often since they can be broken down into segments which for each segment a specific value can be checked. This can be useful to detect patterns. Then by judging these patterns and giving them values the system can start predicting the value that a pattern represents for itself.

The lesson was mostly about how to handle data in a mathematical way so the computer can understand it and generate data that can be used to make decisions.

### What are bitmaps

Bitmaps are an array of data that describe the pixels in an image, for instance JPG, PNG, etc. use bitmaps. The other way to display images that is often use is by using vectors, often used by SVG.

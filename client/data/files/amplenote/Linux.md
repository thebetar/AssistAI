Linux is the brainchild of Linus Torvalds who after using Windows and MacOS decided to create his own Unix based operating system for himself to use. After working on it for a while and asking the community for some feedback it started to gain popularity after which Linus Torvalds open sourced the operating system. Since then a lot of developers have contributed to the open source code base of Linux and made it a preferred operating system for many developers. While most developers do not use base Linux they do use operating systems which are based on Linux like Arch, Debian or Fedora. The most popular operating sytems are even based on operating systems that are based on Linux, like [Ubuntu](Ubuntu.md) and [Linux mint](Linux%20mint.md).

While Linux would not be advisable for non tech savvy users (arguably Ubuntu and linux mint can also be used by non techincal people) it offers great customisation options. This is the main reason that most developers LOVE Linux. Since it is open source and there are many distributions to choose from there exists a distribution that fits the developer's needs out there or at least close to it.

Linux is also the most used operating systems on server, for it's simplicity and flexibility. It is very important to know basic [Unix and Linux commands](Unix%20commands.md) when using linux.

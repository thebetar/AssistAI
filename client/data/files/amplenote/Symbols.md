# Symbols

- $\\nabla$ = gradient of scalar function

- $\\epsilon$ = infinitesimal

- $x \\in y$ = x is element in set y

- $x \\cap y$ = intersection between set $x$ and set $y$

- $x \\cup y$ = union of set $x$ and set $y$

- $x \\subset y$ = set $x$ is a subset of set $y$

- $(f \\circ g)(x)$ = composition function where the result of $g(x)$ is passed as an argument to $f(x)$ this basically means $f(g(x))$

- $\\Delta x$ = value that is added to a value $x$ and decreased to calculate a slope or derivative

- $\\delta$ = lowercase delta describe the change a function performs to it’s parameter.

- $\\theta$ = annotates an angle in degrees

- $\\pi$ = pi describe a fixed value that is used to calculate the area or circumference of a circle

- $e$ = euler’s number which is a fixed value that is used to calculate a logarithmic growth of a value

- $\\Omega$ = describes all the possible ways a combination can take place

- $\\equiv$ = is used to display that two functions have equivalent behaviour

- $\\approx$ = means approximately a value, so close to the answer but not exactly

- $\\sum$ = a summation which means that the answers from the function after it are repeated $x$ times.

- $\\min$ the minimum of a set so $\\min({10, 3, 5, 20}) = 3$ since 3 is the smallest value

- $\\max$ is the maximum of a set so $\\max({10,3,5,20}) = 20$ since 20 is the largest value

- $\\partial$ is the symbol for a partial derivative

- $\\phi$

- $ln$ is the natural logarithm which is used to calculated how a exponential values grows it is best described as $ln(x) = y$ and $e^y = x$ where the value can be calculated in reverse.

- $\\tau$ is equal to the magnitude of a step

- $x : x > 0$ the colon here means a set such that. In this example it will create a set where all values are higher than $0$

- $\|x\|$ is the absolute value of $x$ this means the distance of the number line instead of the value of the number so if $x = -6$ the absolute value is $6$

- $\\lfloor x \\rfloor$ is the floor function this floors the value to the nearest whole number below the decimal number so $\\lfloor 3.7 \\rfloor = 3$

- $\\lceil x \\rceil$ is the ceiling function this does the opposite of the floor function and increases the decimal number to the nearest whole number so $\\lceil 3.7 \\rceil = 4$

- $\\pm$ means plus-minus relationship this is for instance nice if you want to describe $x$ when $x^2 = 16$ is given since $x$ can both be $4$ and $-4$ which results in $x = \\pm 4$

- $\\vec{v}$ is used to denote a vector (the arrow above the number)

- $\|\|\\vec{v}\|\|$ is used to denote the magnitude of a vector

- $\\eta$

- $\\log_2(8)$ is a way to get what power is needed to makes $2$, $8$ in this example. It is like wanting to know $x$ in $2^x = 7$. In this example $log_2(8) = 3$ since $2 \* 2 \* 2 = 8$.

- $\\log(1000)$ most of the time when log is used without a number it means that the logarithm is taken from the base of 10.

- $\\ln(7.389)$ is the notation for a logarithm with base $e$ which is euler’s number (see above description).

- $x!$ in this example the $!$ stands for the factorial of a number. This means that the calculation is done of the number $x$ where each iteration of plus $x$ is subtracted by one so for $4! = 4 \* 3 \* 2 \* 1 = 24$

## Via strato

### Requirements

- Foto's publiek beschikbaar

- Files met wachtwoord (vast)

- Zelf een intro text (oplossing bedenken waarin een HTML ingeladen kan worden op de hoofdpagina)

- Fotos in albums via strato te uploaden met folder structure en foto namen `foto-YYYY-MM-DD.jpg`

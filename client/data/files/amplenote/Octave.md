# Octave

The octave language is an open source option from the very popular matlab language. It was created by the GNU project and is mainly used for data science purposes. The language mostly works with vectors and matrices to handle data and also has a lot of plotting functionalities to show the data.

This is also where the power of the language comes in. Similar to R and Matlab it supports matrix operations which makes working with large datasets a lot more efficient.

## Useful functions

These are some useful functions within the octave language

- `sum(A)` the sum functions sums all the values within a matrix and returns a number

- `trace(A)` the trace function sums all the values along the diagonal of a matrix and returns a number

# Quadratics

# Root

When the graph of a function intersects the $x$ line this is called a root. This means that the output of the function is also equal to 0. To find the root of a function the following formula can be used ${-b \\pm \\sqrt{b^2 - 4ac}\\above{1pt}2a}$ this can be used to calculated the root for a function that looks like $ax^2 + bx + c$

# Discriminant

The discriminant of a graph is calculated by using $b^2 - 4ac$. Doing this will result in a number which if

- it is positive means two real roots

- it is zero means two complex roots

- it is negative means there is one real rool

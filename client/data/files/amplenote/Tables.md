# Tables

In computer science the concept of table exists for storing data. This data consists of a row that has certain values with types. Each value within the item is represented by a column in that row.

## Cross product

When creating the cross product of two tables. The two tables are joint together so every row will match with one other row and the columns will be added to each other increasing the amount of columns and rows.

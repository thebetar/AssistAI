# Distributions

Within math and especially probability theory the word “distribution” is used to desscribe how the values of a dataset are distributed. In most cases this is described by the normal distibution which looks like a bell curve like this

![](local://361a515f-5e82-4469-a1ce-24b7b53397d1)

This is because if you generate a random value let’s say $X$ and you add $n$ samples of this random variable $X$ so a set of ${X_1, …, X_n}$. What is meant by this is if you add lets say $n = 8$ values together and create $\\infin$ amount of samples of this since the random values can be positive or negative this distribution will be biggest around 0 since the chance of values being positive and negative are even.

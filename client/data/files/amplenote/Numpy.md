Numpy is a well known python library used to do data manipulation and vector and matrix operations. The big advantage from using numpy is that it was written in C which is a lot more efficient than python. This makes it so when using numpy to do a large number of basic iterations the speed increase is very noticable.

From an article found on [geeksforgeeks](https://www.geeksforgeeks.org/python-lists-vs-numpy-arrays/) the measured speed difference is

```
Time taken by Lists to perform multiplication: 0.07256507873535156 seconds 
Time taken by NumPy Arrays to perform multiplication: 0.006612300872802734 seconds
```

Which is over 100 times faster!

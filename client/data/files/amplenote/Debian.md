A lot of distros are based on Debian. This is because the core value of Debian is very important, reliability. While more diehard linux fans can fix most of the problems that can occur a lot of user prefer a system that is stable and has a low risk of failing after an update. This has made debian the base of most very popular linux distros, like [linux mint](Linux%20mint.md), [ubuntu](Ubuntu.md), pop_os, etc.

Debian used to be known for it's stability but still being hard to use for beginners, this has recently changed however with newer versions. While it is still recommended to start with either Ubuntu or Linux mint which are debian based but provide some more default programs which are nice to have. After playing around with these for a while it is definetely a good idea to try debian as well since it removes some of the bloat.

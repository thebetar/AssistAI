# Programming+ languages

There are a lot of programming languages out there, some more perfect than others. Each was created to serve a specific purpose well, but also be able to do other things. Within the world of IT a lot of people use mostly one language and swear by that language. See lanuages like Java, C# and typescript. This of course has some merit since if you are very good at one language you will probably make a better program more quickly with that language. But that misses the point and the best programmers know at least a handful of languages. Like said in this page each has their own function and here you will find a list on what the advantage are of each.

## **javascript / typescript**

Javascript and typescript are a lot alike since typescript is a superset of javascript which adds some extra features like typing. Javascripts biggest strength is that it is easy to pick up, has a huge community behind it and is very flexible. Every problem can be solved with javascript and in a lot of instances this is fine. But Javascript falls short in solutions that rely on performance

## **Python**

While Python is not necessarily more performant than javascript it has some nice features which javascript misses like typing. The biggest advantage for python is that it is even easier to pick up, hence why it is taught in so many univeristy courses nowadays and it has a lot of really useful libraries which are written in C that make the computing parts (mostly in machine learning related programs) very fast.

## **[Go](Go.md)**

Go fills up a space between the lowest level language that are most performant like C and very easy and useful languages like javascript and python. This was also the idea behind the creation of Go. A language that is easy to pick up while also delivering performance that python and javascript just can’t bring. While it is slower than programs written in C or rust it is still performant enough for most use cases where javascript and python fall short.

## **[C](C.md) / C++**

While the syntax of C and C++ is different they do both bring the same advantage to the table. If a program needs to be truly performant and small in size one of the best ways to go is C or C++ this is because most other languages are built on top of C so think of it like cutting away the fat. This of course is not 100% the truth but it is a good way to think about it. While C and C++ are the languages operating systems and a lot of things that are used daily are written in a usecase where Go does not cut it is likely not going to come across.

## [Octave](Octave.md)

The octave language is an open source option from the very popular matlab language. It was created by the GNU project and is mainly used for data science purposes. The language mostly works with vectors and matrices to handle data and also has a lot of plotting functionalities to show the data.

# Markup languages

Aside from programming languages there also exist markup languages the most popular example of this is **html**. The main ones to know are

## **html**

html is used to primarly define the structure of a website and where each element lives within the page. While HTML can also be used to store data and other things.

## **[markdown](Markdown.md)**

Markdown is used primarly to write documentation and is a very easy and clean way to write documentation in a small file size, while it does not offer much customisation it is the preferred way for programmers to write documentation

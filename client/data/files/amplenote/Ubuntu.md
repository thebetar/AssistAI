Ubuntu is the most popular Linux distro out there. It is very beginner friendly with the GNOME graphical interface. It is built on top of Debian and uses the `APT` and `snap` package managers.

The idea behind Ubuntu is usability and reliability. This is why it has become so popular. The interface looks familiar and is easy to use while giving the full linux experience.

Since 2020 however Ubuntu has received more and more criticism because it's migration from `apt` to `snap`. And it's `PRO` model where payment is required to receive all the security updates.

A big advantage ubuntu still has over other distros however is that because of it's popularity there are many guides and a big community to help once the user runs into trouble. Also every program that has a program for linux will most likely have a specific guide on how to install or a specific package for ubuntu while sometimes it does not even have this for the underlying OS which is [Debian](Debian.md).

This makes it hard to recommend other distros over ubuntu for beginners since there are many guides and packages already available for ubuntu.
